# လုပ်ဆောင်ချက်များ

သင့်ဆိုင်ကြီးထွားလာစေရန် ကူညီပေးမည့် FOM ၏ လုပ်ဆောင်ချက်များကို လေ့လာပါ။

- **Order Manager**: သင့်အရောင်းအားလုံးကို တစ်နေရာတည်းတွင် စီမံခန့်ခွဲနိုင်သော ဒက်ရှ်ဘုတ်။
- **Customer CRM**: ဝယ်သူတိုင်း၏ မှတ်တမ်းကို အလိုအလျောက် သိမ်းဆည်းပေးခြင်း။
- **Deli Integrations**: အမြန်ချောပို့လုပ်ငန်းများအတွက် ပါဆယ်လ်လိပ်စာများကို အက်ပ်အတွင်းမှ တိုက်ရိုက်ထုတ်ယူနိုင်ခြင်း။
