# စတင်အသုံးပြုရန်

FOM မှ ကြိုဆိုပါတယ်! သင့်ဆိုင်ကို စတင်အသုံးပြုနိုင်ရန် ရိုးရှင်းသော အဆင့် (၃) ဆင့်ကို လုပ်ဆောင်လိုက်ပါ။

## ၁။ အကောင့်ဖွင့်ခြင်း
[getfom.com](https://getfom.com) တွင် သင့်အီးမေးလ် သို့မဟုတ် Facebook အကောင့်ဖြင့် စာရင်းသွင်းပါ။

## ၂။ Facebook Page ချိတ်ဆက်ခြင်း
**Settings > Facebook** သို့သွား၍ "Connect Page" ကို နှိပ်ပါ။ ၎င်းသည် FOM ကို သင့်မတ်ဆေ့ချ်များကို မြင်နိုင်ခွင့် ပေးပါလိမ့်မည်။

## ၃။ ကုန်ပစ္စည်းများ ထည့်သွင်းခြင်း
အော်ဒါများကို အလွယ်တကူ သိမ်းဆည်းနိုင်ရန် **Products** တက်ဘ်တွင် သင့်ဆိုင်ရှိ ပစ္စည်းစာရင်းများကို စတင်ထည့်သွင်းပါ။
