# နိဒါန်း

**FOM Order Manager** ၏ တရားဝင်အသုံးပြုပုံ လမ်းညွှန်မှ ကြိုဆိုပါသည်။ FOM ကို Facebook Messenger မှတစ်ဆင့် အရောင်းအဝယ်ပြုလုပ်နေသော မြန်မာနိုင်ငံရှိ အွန်လိုင်းစျေးသည်များအတွက် သီးသန့်ရည်ရွယ်၍ ထုတ်လုပ်ထားခြင်း ဖြစ်ပါသည်။

## ဘာကြောင့် FOM ကို သုံးသင့်တာလဲ?

Facebook Messenger မှတစ်ဆင့် အော်ဒါများကို လက်ခံရယူခြင်းသည် အလုပ်ရှုပ်စေပါသည်။ ဆိုင်ကြီးထွားလာသည်နှင့်အမျှ ဝယ်သူ၏လိပ်စာ၊ ငွေပေးချေမှုအခြေအနေနှင့် ပစ္စည်းပို့ဆောင်မှုများကို စီမံခန့်ခွဲရန် ခက်ခဲလာပါသည်။ FOM သည် ထိုအခက်အခဲများကို စနစ်တကျ လုပ်ဆောင်နိုင်သော လုပ်ငန်းစဉ်တစ်ခုအဖြစ် ပြောင်းလဲပေးပါသည်။

### အဓိက လုပ်ဆောင်ချက်များ

- **အော်ဒါများကို အလိုအလျောက် သိမ်းဆည်းခြင်း**: မတ်ဆေ့ချ်များကို တစ်ချက်နှိပ်ရုံဖြင့် စနစ်တကျ အော်ဒါများအဖြစ် ပြောင်းလဲနိုင်ခြင်း။
- **ဝယ်သူမှတ်တမ်း သိမ်းဆည်းခြင်း**: ပုံမှန်ဝယ်ယူနေသော ဝယ်သူများ၏ အချက်အလက်နှင့် ပစ္စည်းပို့ဆောင်မှု ဦးစားပေးများကို မှတ်သားထားနိုင်ခြင်း။
- **ပို့ဆောင်မှု စီမံခန့်ခွဲခြင်း**: မြန်မာနိုင်ငံရှိ ထိပ်တန်း အမြန်ချောပို့လုပ်ငန်းများနှင့် ချိတ်ဆက်ဆောင်ရွက်နိုင်ခြင်း။
- **အရောင်းစာရင်း အကျဉ်းချုပ်**: သတ်မှတ်ထားသော ကာလအလိုက် အရောင်းစာရင်းများကို အချိန်နှင့်တပြေးညီ ကြည့်ရှုနိုင်ခြင်း။
