## Product stance

FOM is designed around Facebook-first shops that need a reliable operational system before they
need complex automation.

- Manual input still matters because it keeps the workflow fast and locally understandable.
- Daily visibility matters more than broad ERP scope.
- Access control and plan gating are already part of the product model, even while the platform is
  still expanding.
