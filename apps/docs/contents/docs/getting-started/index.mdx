## What is available today

- A docs-first UI built from the Vercel `Documents` starter structure.
- Searchable markdown and MDX content.
- A seeded `User Manual` section for operational guidance.
- Reference pages pulled from the platform's existing documentation set.

## How to use this site

<Step>
  <StepItem title="Start with the user manual">
    If you are running a shop day to day, begin with the operator-focused pages under
    `User Manual`.
  </StepItem>
  <StepItem title="Check API coverage">
    If you are evaluating integration work, move to `API & Integrations` to see the
    current backend surface and the future direction.
  </StepItem>
  <StepItem title="Use platform reference for deeper context">
    Product planning, architecture, roadmap, and security topics live under
    `Platform Reference`.
  </StepItem>
</Step>

<Note type="warning" title="Current scope">
  Third-party integration APIs are still a future plan. The API section currently documents the
  backend surface and the response conventions that will support that future work.
</Note>
